Design Guidelines

  • Grid System: 24×24px canvas with ~20×20px active area
  • Stroke Weight: Consistent 2px for all elements
  • Line Caps: Round caps and joins for friendly appearance
  • Minimum Size: 16px for UI controls, 20-24px recommended
  • Clearspace: Maintain x-height clearspace around wordmark
  • Accessibility: Include meaningful titles and aria-labels
  • Theming: Uses currentColor for automatic theme integration
  • Tree-shaking: Import only needed icons for optimal bundle size
